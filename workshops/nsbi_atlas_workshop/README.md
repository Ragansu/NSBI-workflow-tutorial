# Tutorial for NSBI workshop in Munich

Installation
---

- Clone the GitHub repository locally using `git clone git@github.com:iris-hep/nsbi-lhc-toolkit.git`
- Run `pixi install -e nsbi_env` if you are using CPU or Mac and `pixi install -e nsbi_env_gpu` if you have access to CUDA-supported GPU.
- Go to the tutorial directory `workshops/nsbi_atlas_workshop/` and start running the notebooks!
